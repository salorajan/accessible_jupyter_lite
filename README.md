\# Accessible Jupyter Lite Pro



Accessible Jupyter Lite Pro is a browser-based Python environment designed from the ground up for screen reader compatibility, specifically optimized for NVDA and JAWS. It functions as a crossbreed between a standard Jupyter Notebook and a MATLAB/Octave command window, making it ideal for math education, including Algebra and Calculus courses.



\## Key Accessibility Features



\* \*\*Streamlined Interface:\*\* Replaces complex, multi-cell layouts with an intuitive, accessible design.

\* \*\*Accessible Cell Selection:\*\* Utilizes a custom ARIA list for code cell selection, ensuring full compatibility with screen reader navigation modes instead of relying on standard combo boxes.

\* \*\*Dedicated Uploads:\*\* Features a distinctly labeled, separate button for notebook uploads to avoid navigation confusion.



\## Keyboard Shortcuts



The environment supports standard notebook execution shortcuts to provide a natural and familiar workflow:



\* \*\*Execute Code:\*\* Press `Shift + Enter` or `Control + Enter` to run the current input. 



\## Deployment



This project is hosted using GitHub Pages. The interface runs entirely in the browser using Pyodide, requiring no server-side Python installation.
