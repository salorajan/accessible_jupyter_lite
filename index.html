Accessible Jupyter Lite Pro

1. Application Introduction

Welcome to Accessible Jupyter Lite Pro. This application is a fully accessible, browser-based Python environment tailored for mathematics education. Its primary advantage is providing a seamless hybrid between a traditional Jupyter notebook and a standard command window. All processing happens locally on your machine, ensuring complete privacy with no server installation required.

Quick Start: You do not need to write code from scratch. Simply activate the "Upload File" button below to load an existing Python script (.py) or Jupyter Notebook (.ipynb), and then press "Run Program" to execute it. We highly recommend uploading the provided test1.py file to see the computational features in action.

Essential Keyboard Shortcuts:

  • Press Alt + 1 to jump directly to the Program Window (Code Editor).
  • Press Alt + 2 to jump directly to the Command Window (Terminal Output).
  • Press Alt + 3 to jump directly to the Figure Window (Visual Plots).
  • Press Shift + Enter or Control + Enter from anywhere to execute your code.
3. Expand Application Manual and Module Commands

3.1. How to Use This Application

  1. Upload: Activate "1. Upload File" to load `.py` scripts or `.ipynb` Jupyter notebooks directly into the editor.
  2. Edit: Write your code in the "Program Window".
  3. Execute: Press `Shift+Enter` or `Control+Enter` from anywhere, or activate "2. Run Program" to execute the code.
  4. Review: Check the "Command Window" for text output and errors. Check the "Figure Window" for generated plots.
  5. Export: Use the download buttons in the toolbar to save your terminal text or generated figures to your local machine.

3.2. Global Keyboard Shortcuts

  • Alt + 1 : Move focus directly to the Program Window (Code Editor).
  • Alt + 2 : Move focus directly to the Command Window (Terminal).
  • Alt + 3 : Move focus directly to the Figure Window (Plots).
  • Shift + Enter or Ctrl + Enter : Execute the Python code currently in the Program Window.

3.3. NumPy Core Commands (Array Mathematics)

  1. np.array([1, 2, 3]) : Create a standard numerical array.
  2. np.linspace(0, 10, 100) : Generate 100 evenly spaced numbers between 0 and 10.
  3. np.arange(0, 10, 2) : Generate numbers from 0 to 10 with a step size of 2.
  4. np.zeros((3, 3)) : Create a 3 by 3 matrix filled with zeros.
  5. np.ones(5) : Create an array of five ones.
  6. np.sin(x) : Calculate the trigonometric sine for all elements in array x.
  7. np.cross(vec1, vec2) : Compute the cross product of two vectors.
  8. np.dot(vec1, vec2) : Compute the dot product of two arrays/vectors.
  9. np.mean(data) : Calculate the mathematical average of an array.
  10. np.std(data) : Calculate the standard deviation of an array.

3.4. SymPy Core Commands (Symbolic Mathematics)

  1. x, y = symbols('x y') : Define x and y as symbolic variables.
  2. diff(sin(x)**2, x) : Calculate the derivative of a function with respect to x.
  3. integrate(x**2, x) : Calculate the indefinite integral of x squared.
  4. integrate(x**2, (x, 0, 1)) : Calculate the definite integral from 0 to 1.
  5. limit(sin(x)/x, x, 0) : Calculate the mathematical limit as x approaches 0.
  6. solve(x**2 - 4, x) : Solve an algebraic equation for x.
  7. expand((x + 1)**2) : Expand a polynomial expression algebraically.
  8. factor(x**2 - 2*x + 1) : Factorize a mathematical expression.
  9. simplify(sin(x)**2 + cos(x)**2) : Simplify an expression to its most basic form.
  10. Matrix([[1, 2], [3, 4]]) : Construct a symbolic mathematical matrix.

3.5. Matplotlib Core Commands (Data Plotting)

  1. plt.plot(x, y) : Create a standard 2D line graph.
  2. plt.scatter(x, y) : Create a scatter plot of individual data points.
  3. plt.bar(categories, values) : Create a vertical bar chart.
  4. plt.title("My Graph") : Add a descriptive title to the top of the plot.
  5. plt.xlabel("Time (s)") : Label the horizontal X-axis.
  6. plt.ylabel("Velocity") : Label the vertical Y-axis.
  7. plt.grid(True) : Enable the background grid lines for easier reading.
  8. plt.legend() : Display a legend identifying different plotted lines.
  9. plt.xlim(0, 10) : Manually set the minimum and maximum limits for the X-axis.
  10. plt.show() : Render the final plot (Automatically mapped to the Figure Window).

3.6. SciPy Core Commands (Optimization & Analytics)

  1. linprog(c, A_ub, b_ub, bounds) : Solve a linear programming problem to minimize an objective function subject to linear equality and inequality constraints.
  2. milp(c, integrality, constraints, bounds) : Solve a mixed-integer linear programming problem where some variables are restricted to integer values.
  3. differential_evolution(func, bounds) : Find the global minimum of a multivariate function using stochastic population-based optimization.
  4. optimize.root(func, x0) : Find the root of a vector function starting from an initial guess x0.
  5. integrate.quad(func, a, b) : Compute a definite integral numerically from a to b.
